## How S.A.U.W. works

```mermaid
flowchart LR
  A[Launch] --> B[Dark Magic];
  B --> C{Math.random < 0.5f};
  C --false--> E[You are lucky];
  C --true--> D[Crash];
  D --> F[Open issue] --> A;
```
