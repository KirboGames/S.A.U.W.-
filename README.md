## How to Run
* Windows:
```shell
git clone https://github.com/KirboGames/S.A.U.W.-
cd S.A.U.W.-
gradlew desktop:dist
cd desktop/build/libs
java -jar desktop-1.0.jar
```
* Linux:
```shell
git clone https://github.com/KirboGames/S.A.U.W.-
cd S.A.U.W.-
bash gradlew desktop:dist
cd desktop/build/libs
java -jar desktop-1.0.jar
```
