# INSTRUMENT

Данный тип предмета позволяет создать свой инструмент.

```json5
{
  //...
  "configuration": {
    //...
    "type": "INSTRUMENT",
    "instrument_type": "PICKAXE"
  }
}
```

`instrument_type` - тип инструмента

Есть 3 типа инструментов

* `PICKAXE` - Кирка
* `AXE` - Топор
* `SHOVEL` - Лопата
